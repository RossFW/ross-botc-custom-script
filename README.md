# Friends & Foes — Custom BotC Script

A homebrew Blood on the Clocktower script. View the live sheet at:
**https://rossfw.github.io/ross-botc-custom-script/**

## Editing (for the author)
The sheet is editable locally with autosave to `data.json`:
```
python3 server.py
# open http://localhost:8770/
```
Edit inline, hit **Save Changes**, then commit & push `data.json` to update the live site.

> The live GitHub Pages site is static (no save server), so it's view-only for visitors.

## Export to the official Script Tool
Use the **🎲 Export for Script Tool** button to download a JSON you can import at
[script.bloodontheclocktower.com](https://script.bloodontheclocktower.com/).
